列选择

概述

列选择可以用来选择一个文件中的矩形区域。 列选择不通过一个单独的模式运作，而是使用多重选择。

你可以添加选取来选择文本的多个代码块或者减少选取去掉一个代码块。

使用鼠标

不同平台使用不同的鼠标键和键盘键:

OS X

  • 鼠标左键 + Option 或者: 鼠标中键
  • 添加至选区: Command
  • 从选区减少: Command+Shift

Windows

  • 鼠标右键 + Shift 或者: 鼠标中键
  • 添加至选区: Ctrl
  • 从选区减少: Alt

Linux

  • 鼠标右键 + Shift
  • 添加至选区: Ctrl
  • 从选区减少: Alt

使用键盘

OS X

  • Ctrl + Shift + Up
  • Ctrl + Shift + Down

Windows

  • Ctrl + Alt + Up
  • Ctrl + Alt + Down

Linux

  • Ctrl + Alt + Up
  • Ctrl + Alt + Down
