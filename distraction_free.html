自由模式（沉浸模式）

概述

自由模式（沉浸模式）全屏显示你的文件，你的显示器当中除了文本啥也没有。所有的UI chrome是隐藏的，但是可以访问。沉浸模式可以通过View/Enter Distraction Free Mode菜单项进入。

当处于沉浸模式时，所有的UI chrome (侧边栏,迷你地图,状态栏,等)将被隐藏。你可以通过View菜单选择性的选择部分UI-你的配置将会在下次进入沉浸模式时生效。

定制化

部分配置将会在进入沉浸模式时应用。这些默认的配置位于Packages/Default/Distraction Free.sublime-settings)他们是: 

{
	"line_numbers": false,
	"gutter": false,
	"draw_centered": true,
	"wrap_width": 80,
	"word_wrap": true,
	"scroll_past_end": true
}

你可以定制这些通过编辑Packages/User/Distraction Free.sublime-settings，这可以通过Preferences/File Settings - More 菜单访问到。

上面值得注意的是wrap_width设置。它的值80造成超过80字符时换行。你也许想把它设的高点，或者在窗口宽度是换行（设为0的时候）。
